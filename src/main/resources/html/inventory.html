Vaadin Template Application

His wrapper an static HTML file import.

The template includes the following features:

  • Simple login control
    • Not usable for real applications
  • Route permission annotation
    • @PermissionTab annotation can be used to give different roles permissions to specific views or routes.
  • Theming
    • Themes are stored in "frontend/theme/custom-themes.css" and then imported via @CSSImport
  • Internationalization
    • An implementation of I18NProvider is also included. The translation files are stored in the Java main/resources/translations.
